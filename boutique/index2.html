📋 Spécifications Techniques

Guide complet pour l'implémentation. Architecture, fonctionnalités, et intégrations.

Architecture Fichiers

/boutique/
├── index.html (Page catalogue principal)
├── produit.html (Template page produit individuel)
├── panier.html (Page panier/cart)
├── confirmation.html (Après achat)
├── css/
│ ├── reset.css
│ ├── variables.css (Couleurs, fonts)
│ ├── layout.css (Grid, structure)
│ ├── components.css (Buttons, cards, forms)
│ └── boutique.css (Styles spécifiques boutique)
├── js/
│ ├── products.js (Data JSON produits)
│ ├── cart.js (Logique panier localStorage)
│ ├── filters.js (Filtres catégories)
│ └── stripe.js (Intégration checkout)
└── images/
└── products/ (Photos produits)

Fonctionnalités Clés

🛒 Panier Persistant
localStorage pour sauvegarder panier entre sessions. Survit fermeture navigateur.
🔍 Filtres Live
Filtrage instantané par catégorie sans reload. JavaScript vanilla.
💳 Stripe Checkout
Redirect vers Stripe hosted checkout. Secure, PCI compliant.
📱 Responsive
Mobile-first design. Grid adaptatif. Touch-friendly sur tablettes.
📧 Email Auto
Cloudflare Email Workers pour confirmations commandes automatiques.
🌐 Bilingue FR/EN
Toggle langue comme site actuel. Data attributes pour traductions.
