1. Portée des Services

RVS Production offre les services suivants :

  • Développement de sites web professionnels
  • Corrections et optimisation de sites existants
  • Conformité Loi 25 (Québec)
  • Hébergement web sécurisé
  • Maintenance et support technique (optionnel)

2. Gestion des Noms de Domaine

2.1 Propriété du Domaine

Le CLIENT demeure propriétaire légal de son nom de domaine en tout temps. RVS Production n'acquiert AUCUN droit de propriété sur le domaine du client.

2.2 Trois Options Disponibles

OPTION A - Client conserve son domaine existant

  • Le CLIENT garde son domaine actuel
  • Le CLIENT fournit accès administrateur à RVS Production
  • Aucuns frais de gestion de domaine
  • Le CLIENT paie directement son renouvellement annuel à son registraire

OPTION B - Client achète un nouveau domaine

  • Le CLIENT achète son propre domaine (chez GoDaddy, Namecheap, etc.)
  • Le CLIENT fournit accès administrateur à RVS Production
  • RVS Production guide gratuitement le CLIENT dans l'achat
  • Aucuns frais de gestion de domaine
  • Le CLIENT paie directement son renouvellement annuel à son registraire

OPTION C - RVS Production gère l'acquisition/transfert

  • RVS Production recherche, achète ou transfère le domaine
  • Frais : 125$/heure (temps réel de gestion)
  • Frais de transfert/acquisition : coût réel du domaine facturé au CLIENT
  • Frais de gestion annuels : 50$/an après la première année

2.3 Accès Administrateur Requis

Pour tous les projets, RVS Production requiert :

  • Accès admin au compte du registraire (GoDaddy, Namecheap, etc.)
  • OU délégation DNS complète vers les serveurs RVS Production
  • OU transfert temporaire des codes EPP/Auth pour configuration

Ces accès permettent :

  • Configuration DNS
  • Connexion au serveur d'hébergement
  • Configuration des emails professionnels
  • Gestion des certificats SSL
  • Modifications techniques au besoin

2.4 Récupération de Domaines Existants

Si le CLIENT demande à RVS Production de récupérer/transférer un domaine depuis un ancien fournisseur ou développeur :

Frais applicables : 125$/heure + frais de fournisseur de domaine(facturation au temps réel)

Ces frais couvrent :

  • Recherche et identification du registraire actuel
  • Communications avec l'ancien fournisseur
  • Récupération des codes de transfert (EPP/Auth codes)
  • Procédures de transfert ou récupération de contrôle
  • Résolution de conflits ou blocages administratifs
  • Vérification et tests post-transfert

Temps estimé typique : 1-3 heures (selon complexité)

2.5 Fin de Contrat - Retour du Domaine

À la fin du contrat avec RVS Production :

Le CLIENT récupère immédiatement :

  • Accès admin complet à son domaine
  • Tous les codes et credentials nécessaires
  • Documentation de configuration
  • Fichiers du site web (code source complet)

Frais de transfert/transition : 125$/heure + frais de fournisseur de domaine

Ces frais couvrent :

  • Transfert du domaine vers le nouveau fournisseur du CLIENT
  • Reconfiguration DNS vers nouvelle destination
  • Exportation et remise des fichiers
  • Documentation technique pour transition
  • Support durant la période de transition (jusqu'à 2 heures)

Temps estimé typique : 1-2 heures

Note : Ces frais sont facturés UNIQUEMENT si le CLIENT demande l'assistance de RVS Production pour la transition. Le CLIENT peut aussi gérer lui-même la transition sans frais additionnels s'il le souhaite.

3. Hébergement Web

3.1 Service Inclus

Hébergement - Deux tarifs selon le type de projet :

A Site développé par RVS Production : 120$/an

(Refonte complète ou nouveau site créé par RVS Production)

B Hébergement site existant seulement : 255$/an

(Site non développé par RVS Production)

Les deux tarifs incluent :

  • Hébergement sécurisé sur Cloudflare Pages
  • Certificat SSL (https) automatique
  • Bande passante illimitée
  • CDN mondial (vitesse optimale)
  • Protection DDoS basique
  • Support technique inclus
  • Sauvegardes automatiques quotidiennes

Note importante : Le tarif réduit (120$/an) s'applique uniquement aux sites dont RVS Production a assuré le développement complet (refonte ou création). Pour l'hébergement de sites existants développés par d'autres, le tarif standard de 255$/an s'applique.

4. Maintenance et Support

4.1 Maintenance Optionnelle

Formule de base : 200$/mois

  • Jusqu'à 2 heures de modifications par mois
  • Mises à jour de contenu (textes, images)
  • Corrections de bugs mineurs
  • Support par email (réponse sous 48h)

Formule avancée : 400$/mois

  • Jusqu'à 4 heures de modifications par mois
  • Surveillance proactive
  • Sauvegardes hebdomadaires additionnelles
  • Support prioritaire (réponse sous 24h)
  • Optimisations continues

4.2 Support Ponctuel

Si le CLIENT ne prend pas de forfait maintenance :

  • Tarif horaire : 125$/heure
  • Facturation à l'heure près
  • Minimum facturable : 1 heure

RVS Production n'est pas responsable de :

  • Pertes de revenus dues à interruptions de service
  • Dommages indirects ou consécutifs

  • Violations de droits d'auteur dans le contenu du CLIENT

Responsabilité maximale limitée au montant payé pour le service dans les 12 derniers mois.

RVS Production s'engage à :

  • Protéger les informations confidentielles du CLIENT
  • Ne pas divulguer les accès et credentials
  • Respecter la Loi 25 (Québec) sur la protection des données
